This program will convert an Imperator: Rome save game to a Crusader Kings III save game.

----
IMPORTANT LINKS
----

Github project: https://github.com/ParadoxGameConverters/ImperatorToCK3
Development build: https://github.com/ParadoxGameConverters/ImperatorToCK3/releases/tag/DevelopmentRelease

----
REQUIREMENTS
----
Imperator: Rome - version 1.4 or higher

-----
INSTRUCTIONS
-----
1. Run ConverterFrontend.exe.
2. Fill out all fields and keep clicking Next.
