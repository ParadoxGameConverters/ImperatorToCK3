This program converts an Imperator: Rome save game to a Crusader Kings III mod.

----
IMPORTANT LINKS
----

GitHub project: https://github.com/ParadoxGameConverters/ImperatorToCK3

----
REQUIREMENTS
----
Imperator: Rome - version 2.0
Crusader Kings III - version 1.12

-----
INSTRUCTIONS
-----
1. Run ConverterFrontend.exe.
   Watch this video to see how to use the converter: https://www.youtube.com/watch?v=KzqAHH-l0Wc
2. Fill out all paths and set up any options. Then click "Convert Save".
